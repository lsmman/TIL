
# Python env

### 아나콘다 설치

- `conda create -n helloML python=3.6`

### pip 업그레이드

- `pip update`
- `pip install --upgrade pip`

### 사용 라이브러리 인스톨
- Anaconda를 설치시 jupyter가 같이 설치되어 주피터 노트북을 바로 사용할 수 있다. 

- `pip install numpy matplotlib pandas scipy`
- `pip install ipython scikit-learn`
- `pip install jupyter`
